# EST Clean - React 웹사이트

EST Clean 전문 청소 서비스 웹사이트를 React로 구현한 프로젝트입니다.

## 기술 스택

- React 18
- React Router DOM
- Vite
- CSS3

## 설치 및 실행

### 의존성 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

### 프로덕션 빌드
```bash
npm run build
```

### 프로덕션 미리보기
```bash
npm run preview
```

## 프로젝트 구조

```
src/
  ├── components/      # 재사용 가능한 컴포넌트
  │   ├── Header.jsx
  │   └── Footer.jsx
  ├── pages/          # 페이지 컴포넌트
  │   ├── Home.jsx
  │   └── About.jsx
  ├── App.jsx         # 메인 앱 컴포넌트 (라우팅)
  ├── main.jsx        # 엔트리 포인트
  └── index.css       # 전역 스타일
public/
  └── images/         # 이미지 파일
```

## 주요 기능

- 반응형 디자인
- SPA (Single Page Application) 라우팅
- 모던한 UI/UX
